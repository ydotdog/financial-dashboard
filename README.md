# Financial Dashboard

宏观金融看板 — 四层分析框架（价格、流动性、基本面、情绪）。

## 功能

- **顶部滚动行情**：实时展示 US10Y、DXY、VIX、S&P 500、黄金、主要汇率等
- **核心指标**：10Y TIPS 实际收益率作为终极锚定指标
- **四层分析框架**：
  - 价格层：10年期美债收益率、美元指数、收益率曲线利差
  - 流动性层：联邦基金利率、SOFR、高收益信用利差
  - 基本面层：宏观经济基本面指标
  - 情绪层：市场情绪与风险偏好指标

## 数据源

- CNBC Quote API（实时行情）
- FRED（美联储经济数据）

## 技术栈

- HTML / JavaScript（前端看板）
- Python（`server.py` 本地服务）
- Shell（`fred_fetcher.sh` 定时数据拉取）

## 使用方式

双击 `start.command` 即可启动本地服务并打开看板。
